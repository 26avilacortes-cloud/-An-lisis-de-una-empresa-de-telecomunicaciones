# -An-lisis-de-una-empresa-de-telecomunicaciones
Este proyecto analiza el comportamiento de consumo de los clientes de la empresa de telecomunicaciones ConnectaTel. . El objetivo principal es identificar patrones de uso y segmentar a los usuarios por edad y nivel de consumo para ofrecer recomendaciones estratégicas que mejoren la rentabilidad y la retención de clientes.
